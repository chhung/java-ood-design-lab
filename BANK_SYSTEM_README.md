# 銀行管理系統 - 使用說明

## 📋 功能介紹

這是一個互動式的文字銀行管理系統，提供以下功能：

### 1. 顯示帳戶餘額
- 顯示所有已開設的帳戶列表
- 顯示每個帳戶的餘額
- 計算並顯示所有帳戶的總餘額
- 標示當前操作的帳戶

### 2. 新開帳戶
- 建立新的銀行帳戶
- 可設定初始存款金額
- 初始金額可為 0（不存款）
- 不允許負數金額

### 3. 存入金額
- 選擇要存款的帳戶（如果有多個帳戶）
- 輸入存款金額
- 顯示存款前後的餘額變化
- 驗證金額必須大於 0

### 4. 領出金額
- 選擇要提款的帳戶（如果有多個帳戶）
- 輸入提款金額
- 檢查餘額是否足夠
- 顯示提款前後的餘額變化
- 驗證金額必須大於 0

### 5. 退出系統
- 結束程式並關閉系統

## 🚀 如何執行

```powershell

cd C:\Data\code\gotogoogle\java-ood-design-lab\src\main\java

# 2. 編譯程式
javac -encoding UTF-8 idv/js/generic/Account.java idv/js/console/Screen.java

```

### 方法 2: 使用 IntelliJ IDEA
1. 在編輯器中打開 `Screen.java`
2. 找到 `main` 方法
3. 點擊行號旁邊的綠色執行按鈕 (▶)
4. 選擇 "Run 'Screen.main()'"

## ⚠️ 錯誤處理

系統會處理以下錯誤情況：

1. **無效的選項**: 輸入不在 1-5 範圍內的數字
2. **格式錯誤**: 輸入非數字的內容
3. **負數金額**: 不允許輸入負數
4. **餘額不足**: 提款金額超過帳戶餘額
5. **無效金額**: 存款或提款金額必須大於 0
6. **無帳戶操作**: 在沒有帳戶時嘗試進行存款或提款

- ✅ 支援多個帳戶管理
- ✅ 自動追蹤當前操作帳戶

- ✅ 友善的錯誤訊息
- ✅ 清晰的餘額變化顯示
- ✅ 總餘額統計功能

## 💡 提示

- 如果只有一個帳戶，系統會自動選擇該帳戶
- 系統會記住最後操作的帳戶（顯示 [當前] 標記）
- 提款時會自動檢查餘額是否足夠
- 所有金額都會顯示到小數點後兩位
